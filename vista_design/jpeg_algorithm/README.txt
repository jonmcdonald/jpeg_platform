To compile and run the original, synthesized C++ code for the IDCT and testbench:
1. Set environment variable CATAPULT_HOME to the <Catapult install directory>/Mgc_home (e.g. /home/student/Catapult_Synthesis_2009a.168/Mgc_home)
2. Make sure a valid g++ for your platform is in your path
3. Type "make gcc"

To synthesize with Catapult:
1. Set environment variable CATAPULT_HOME to the <Catapult install directory>/Mgc_home (e.g. /home/student/Catapult_Synthesis_2009a.168/Mgc_home)
2. Type "make catc"


To remove all temp files:
1. Type "make clean"
